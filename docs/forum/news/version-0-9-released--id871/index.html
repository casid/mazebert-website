Version 0.9 released

Version 0.9 is released to the Google Play Store! It should become available within the next hours. Versions for iOS and Windows will follow on Friday - iOS users can expect the update within a week (depending on Apple's review speed). Both in terms of time and content, this is one of the biggest releases I've done so far for Mazebert. Thank you so much for your patience and all the great input you gave me to make this game better and better. I'm very, very proud! Here comes the changelog... - Cards can be transmuted directly with a swipe gesture - New player skill from LHL that increases your life - Bonus round added, wave master skill removed - End of game screens for victory and defeat - New towers: Nomad's Scarface, Mr. Iron, Miss Jilly and Irish Pub - New set: Mokhnaar's Impatience Wrath - New items: Pair of Wedding Rings - In-game music that loops - Several bugfixes and balancing adjustments You can see the complete changelog here.
